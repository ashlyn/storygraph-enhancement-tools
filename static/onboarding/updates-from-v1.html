New in The StoryGraph Enhancement Tools!

Updates to Library Links

The StoryGraph Enhancement Tools by default now uses the OverDrive platform, rather than Libby. OverDrive requires no additional setup for new or existing users.

If you'd prefer to use the Libby platform, you will need to reconfigure your library name. Log into Libby, select the search tab, and find your library name in the web address. Then come back to the settings, select the Libby option, and enter your library's name.
