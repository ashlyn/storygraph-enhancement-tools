Welcome to The StoryGraph Enhancement Tools!

Features

You will start seeing links to search for books to checkout at your local library and to purchase via Amazon, eBooks.com, and Kobo when you use The StoryGraph.

If you would prefer to turn off any of these features, click the icon above to open up the settings, where you can individually toggle each feature on or off.

Library

By default, this extension will search for books on the OverDrive platform. If you'd prefer to use the Libby platform, log into Libby, select the search tab, and find your library name in the web address. Then come back to the settings, select the Libby option, and enter your library's name.

Amazon

If you live outside of the United States and want to use one of the regional Amazon addresses (not Amazon.com), select your country in the dropdown.

Other Storefronts

Links to search the Kobo and eBooks.com store can also be added to the Buy section.
